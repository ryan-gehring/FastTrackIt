## Overview
This project is intended to pull auctions and items from the website BidFTA. It is intended to scape all auctions by location and store those resulting auction details and items in a database.

## Design
The database will store auction details along with all auction items.

## Components
*Conductor - Primary user interaction for defining settings and managing actions  
*Crawler - Contains all required functions to navigate and pull data from the website  
*Database - Database setup and maintenance functions
